#  MAISON — Responsive Furniture Store Landing Page

A responsive landing page for a premium online furniture store, built with **HTML5**, **CSS3**, and **Bootstrap 5**.

---

##  Preview

> Open `index.html` in your browser to view the page. It is fully responsive and works across desktop, tablet, and mobile devices.

---

##  Project Structure

```
maison-store/
├── index.html       # Main HTML structure
├── style.css        # Custom CSS styles & responsive rules
└── README.md        # Project documentation
```

---

## Features

- **Fixed Header / Navbar** — Sticky navigation with a frosted-glass backdrop, dropdown category menu, and phone contact info
- **Hero Section** — Full-viewport background with parallax effect, animated headline, eyebrow label, and dual CTA buttons
- **Stats Bar** — Trust-building strip showing clients, products, countries, and rating
- **Products Section** — 8 product cards each with image, name, description, price, badge (New / Best Seller / Sale), and Shop Now button
- **About Section** — Brand story with floating year badge, feature icons, and a call-to-action
- **Footer** — Brand description, quick links, contact info, and social media icons
- **Fully Responsive** — Tested across desktop, tablet, and mobile screen sizes

##  Design Choices

| Decision | Choice | Reason |
|---|---|---|
| **Display Font** | Cormorant Garamond | Elegant serif gives a luxury, editorial feel |
| **Body Font** | DM Sans | Clean and highly readable at small sizes |
| **Accent Color** | `#B8975A` (gold) | Signals premium quality and warmth |
| **Background** | `#FAF8F5` (warm off-white) | Softer than pure white; easier on the eyes |
| **Hero BG** | Parallax image + dark overlay | Creates depth and draws attention to the headline |
| **Card Hover** | Lift + image zoom | Subtle interaction feedback without being distracting |
| **Navbar** | Fixed + backdrop blur | Keeps navigation accessible while scrolling |

##  Responsive Breakpoints

| Breakpoint | Target |
|---|---|
| `≥ 992px` | Desktop layout |
| `≤ 991px` | Tablet — adjusted navbar, centered About section |
| `≤ 768px` | Mobile — stacked hero buttons, reduced image heights |
| `≤ 480px` | Small mobile — smaller text, full-width buttons, scroll parallax disabled |

---

##  How to Run

1. Clone or download the repository
2. Make sure `index.html` and `style.css` are in the **same folder**
3. Open `index.html` in any modern browser

No build tools or dependencies required — it runs entirely in the browser.
